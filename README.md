# Tribute Page Project

This is a simple, responsive Tribute Page built as part of a web development lab. The goal was to create a static web page honoring a notable individual while fulfilling a specific set of user stories to ensure HTML/CSS best practices and responsiveness.

## 🧾 Project Features

- Semantic HTML5 structure
- Responsive layout using CSS
- External link that opens in a new tab
- Fully passes all 19 required user stories

---

## ✅ User Stories Covered

1. `main` element with `id="main"` containing all content  
2. Title in an element with `id="title"`  
3. Image container with `id="img-div"`  
4. Image inside `#img-div` with `id="image"`  
5. Image caption with `id="img-caption"`  
6. Text content section with `id="tribute-info"`  
7. External link with `id="tribute-link"` using `target="_blank"`  
8. Responsive image with `max-width: 100%` and `height: auto`  
9. Centered image using `display: block` and `margin: auto`
